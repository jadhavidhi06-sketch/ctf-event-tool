NAME
    ctfforge.py - Discover Capture The Flag (CTF) events in India and worldwide

SYNOPSIS
    python ctfforge.py

DESCRIPTION
    CTFForge is a powerful command-line tool that scrapes live CTF events from
    multiple sources (CTFtime API, CTFtime HTML, CTF Hunt), ranks them using
    AI heuristics, and displays them in a beautiful terminal interface with
    clickable links.

FEATURES
    • Live scraping from multiple CTF sources
    • AI-based event ranking (date, participants, location, weight, prizes)
    • Support for single and multi-state selection
    • Shows only ongoing and upcoming events
    • Full event links for direct registration (clickable in terminal)
    • Multiple filter options (upcoming, ongoing, offline, high weight)
    • Multiple sort options (AI ranked, date, participants, weight)
    • Export results to JSON and CSV files
    • Detailed event view with full description
    • Statistics dashboard
    • No API keys required

STEP-BY-STEP USAGE

1. State Selection Mode
   --------------------
   • Multi-state selection (recommended)
   • Single state selection

2. State Selection
   ---------------
   You will see a numbered list of all Indian states and union territories.
   - For multi: Enter numbers separated by commas (e.g., 14,22,27) or type "all"
   - For single: Enter one number

3. Filter Options
   --------------
   1. All CTF Events
   2. Only Upcoming CTF Events
   3. Only Ongoing CTF Events
   4. Only Offline CTF Events
   5. Only High Weight CTF Events (>50)

4. Sort Options
   ------------
   1. AI Ranked (Recommended) - Uses smart heuristics
   2. By Start Date - Earliest first
   3. By Participants Count - Most popular first
   4. By CTF Weight - Highest weight first

5. Live Scraping
   -------------
   The tool will show a progress spinner while fetching live data from:
   • CTFtime API (primary source)
   • CTFtime HTML (detailed info)
   • CTF Hunt (additional source)

6. Results Display
   ---------------
   Events are shown in a ranked table with:
   - Rank (based on AI score or sort order)
   - CTF Name
   - Dates (start → end)
   - Location (Online/Offline with state info)
   - Format (Jeopardy, Attack-Defense, Mixed)
   - Weight (CTFtime weight rating)
   - Participants count
   - Direct Link (clickable in supporting terminals)

7. Statistics Dashboard
   -------------------
   After results, you'll see:
   • Total CTFs found
   • Online CTFs count
   • Offline CTFs count
   • High weight CTFs count
   • Average participants

8. Export Options
   -------------
   • Save to JSON (recommended) - Full event data
   • Save to CSV - Spreadsheet-friendly format
   Filename format: ctfforge_<date>_<time>.json/.csv

9. Detailed Event View
   -------------------
   Option to view complete details of any event including:
   • Full description
   • Organizer information
   • Prize details
   • Status
   • Clickable link

TIPS & BEST PRACTICES

• Always choose "All CTF Events" first time to see everything available.
• Use multi-state selection if you are open to travel.
• Online events are automatically included regardless of state.
• Click the link directly in the terminal (Ctrl+Click on most terminals).
• Use "High Weight" filter to find prestigious CTFs.
• Run the tool frequently as new CTFs are added daily.
• Export to CSV for sharing with your CTF team.
• Use "Ongoing" filter to find CTFs happening right now.

UNDERSTANDING CTF WEIGHT

• Weight 0-25: Beginner/Junior CTFs
• Weight 25-50: Intermediate CTFs
• Weight 50-75: Advanced CTFs
• Weight 75-100: Top-tier International CTFs
• Higher weight = More prestigious and challenging

UNDERSTANDING AI RANKING

The AI ranking considers:
• Days until start (sooner = better)
• Number of participants (popular = better)
• Indian state presence (local = better)
• Currently ongoing (active = better)
• CTF weight (prestigious = better)
• Prize availability (monetary = better)
• Offline format (in-person = better)

TROUBLESHOOTING

• "Playwright not installed" → Run: playwright install chromium
• "Module not found" → Run: pip install httpx playwright rich
• Slow first run → Normal, Chromium is being downloaded
• Scraping warnings → Normal, some sites change layout frequently
• No events found → Try "All CTF Events" + "all" states + no filters
• Links not clickable → Use terminal that supports hyperlinks (Windows Terminal, iTerm2, etc.)
• CTF Hunt errors → Normal, this source is sometimes unavailable
